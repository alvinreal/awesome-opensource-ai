## Add MLOps Experiment Tracking & Monitoring Tools

This PR adds 4 elite-tier open-source MLOps tools to the **MLOps / LLMOps & Production** section:

### Experiment Tracking
- **[Sacred](https://github.com/IDSIA/sacred)** (4,363 stars, MIT) - Python tool to configure, organize, log and reproduce computational experiments with config scopes and MongoDB observers.

### ML Monitoring & Observability
- **[NannyML](https://github.com/NannyML/nannyml)** (2,135 stars, Apache 2.0) - Post-deployment ML monitoring with performance estimation without ground truth.
- **[whylogs](https://github.com/whylabs/whylogs)** (2,814 stars, Apache 2.0) - Privacy-preserving data logging library for ML model monitoring.
- **[Alibi Detect](https://github.com/SeldonIO/alibi-detect)** (2,517 stars, Apache 2.0) - Outlier, adversarial and drift detection for tabular, text, image and time series data.

All projects meet the elite criteria:
- 1000+ GitHub stars
- Active development (commits within last 3 months)
- OSI-approved open source licenses
- Production-ready with good documentation
