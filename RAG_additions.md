# RAG Section Additions - April 2026

## Proposed Additions to "RAG Frameworks & Advanced Retrieval Tools" Section

### 1. cognee
- **[cognee](https://github.com/topoteretes/cognee)** ![GitHub stars](https://img.shields.io/github/stars/topoteretes/cognee?style=social) - Knowledge Engine for AI Agent Memory. Ingest data in any format and continuously learn to provide the right context for AI agents. Features knowledge graph construction, semantic search, and memory management. Apache 2.0 licensed.

**Stats:** 16,592+ stars, Apache 2.0, actively maintained (recent commits in 2025)

### 2. HippoRAG
- **[HippoRAG](https://github.com/OSU-NLP-Group/HippoRAG)** ![GitHub stars](https://img.shields.io/github/stars/OSU-NLP-Group/HippoRAG?style=social) - Neurobiologically inspired long-term memory for LLMs. Novel RAG framework that enables continuous knowledge integration across documents using knowledge graphs and personalized PageRank. NeurIPS'24 and ICML'25. MIT licensed.

**Stats:** 3,381+ stars, MIT, actively maintained (recent activity 2025)

### 3. Verba
- **[Verba](https://github.com/weaviate/Verba)** ![GitHub stars](https://img.shields.io/github/stars/weaviate/Verba?style=social) - The Golden RAGtriever - Community-driven open-source RAG application with end-to-end, user-friendly interface. Powered by Weaviate vector database with modular architecture for customizable RAG pipelines. BSD-3-Clause licensed.

**Stats:** 7,654+ stars, BSD-3-Clause, actively maintained (Verba 0.3.0 released 2025)

---

**Research Notes:**
- All three projects have 1000+ GitHub stars
- All have OSI-approved licenses (Apache 2.0, MIT, BSD-3-Clause)
- All show active development within the last 3 months
- None are currently listed in the repository's RAG section
- RAPTOR was considered but excluded due to inactivity (last commit Sept 2024)
- fastRAG was considered but excluded due to Intel ending maintenance
